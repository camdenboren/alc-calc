# Keybinds

_macOS uses Cmd instead of Ctrl_

| Keybind          | Action                                  |
| ---------------- | --------------------------------------- |
| Ctrl-A           | Select all                              |
| Ctrl-C           | Copy                                    |
| Ctrl-V           | Paste                                   |
| Ctrl-X           | Cut                                     |
| Ctrl-I           | Add an ingredient                       |
| Ctrl-D           | Delete the last ingredient              |
| Ctrl-R           | Delete the current ingredient           |
| Ctrl-Q           | Quit alc-calc                           |
| Ctrl-T           | Toggle theme menu                       |
| Ctrl-N           | Open a new window                       |
| Ctrl-W           | Close the current window                |
| Ctrl-Shift-Space | Show the character palette (macOS only) |
